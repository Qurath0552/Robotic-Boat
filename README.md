# Robotic-Boat
An Arduino-based autonomous robotic boat designed to collect floating plastic waste from water bodies. Powered by a battery and controlled via Bluetooth, it uses DC motors with an L298N driver and a conveyor mechanism to provide a low-cost, eco-friendly solution for reducing marine pollution.
